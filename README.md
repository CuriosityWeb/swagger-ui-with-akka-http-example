# Swagger UI With Akka Http Example

## Configuration

| **Lib** | Version |
|--|--|
| **Scala** | 2.13.8 |
| **Swagger UI** | 4.12.0 |
| **Akka** | 2.6.19 |
| **Akka Http** | 10.2.9 |

## How to run?

    sbt run

Launch your browser and navigate to **localhost:8080**.
